# 💳 Union Bank System

**Union Bank System** is a Java desktop banking application developed with **Apache NetBeans**. It provides a user-friendly interface for managing bank accounts and performing essential banking operations. The system uses **Microsoft Access** as the database, which is ideal for a small-scale banking system. All necessary tables are already created and ready to use.

---

## Features
- Create, read, update, and delete user accounts (CRUD)
- Deposit and withdraw money
- Check account balances
- Simple, intuitive GUI

---

## Technologies
- **Language:** Java
- **GUI Framework:** Swing / JavaFX
- **IDE:** Apache NetBeans
- **Database:** Microsoft Access

---

## Getting Started

### ▶️ Run in Apache NetBeans
1. Open **Apache NetBeans IDE**
2. Click `File → Open Project` and select your project folder
3. Right-click the project in the **Projects pane** → `Run`, or press **F6**

---

### 💻 Run from Terminal

**Step 1 — Compile source files:**
```bash
javac -d build/ src/**/*.java
```

**Step 2 — Run the main class** *(replace with your actual main class path)*:
```bash
java -cp build/ com.unionbank.Main
```

> ⚠️ Make sure the main class contains `public static void main(String[] args)`
